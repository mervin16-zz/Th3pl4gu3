Mauritius Emergency Services Go API

Project information

  • Category: Golang, Gin Gonic, API
  • Project URL: API Endpoint | GitHub Repo

Mauritius Emergency Services is an API that provides a list of emergency phone services in Mauritius. It has been written in Go and Gin Gonic Framework hosted on Prometheus. It is a improvement of the MES written in Python. Current supported languages are English and French.
